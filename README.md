# 🎵 Audio Splitter

A personal tool to separate vocals from background audio in music and video files.

## Features
- Upload audio (mp3, wav, flac, m4a) or video (mp4, mkv, avi, mov, webm)
- Separate vocals from background music
- Choose between Fast or Best Quality model
- Preview tracks before downloading
- Drag & drop support

## Built with
- Python / Flask
- Demucs (Meta AI)
- ffmpeg

## Usage
Run locally with `run.bat` or `py -3.11 -m flask run`
